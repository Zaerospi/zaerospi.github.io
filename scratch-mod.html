Set list block

This mod adds two new blocks to the block palatte, and both are capable of setting lists. One block sets one list to another by selecting them from dropdown menus. The other block sets a list to a string, by selecting a list in a dropdown and putting a formatted string into the other input. Each word in the string becomes an element, or if there is only one word, each character becomes an element.
This mod was made to provide an easily accessible hands on demo for the set list to string block. I have a forum topic on scratch.mit.edu for that block here. The topic is closed to prevent necroposts because the topic is linked to from here and my signature.
